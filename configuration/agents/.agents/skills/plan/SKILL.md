Take guidance from the user and turn it into a plan that can be handed over for implementation.

Use /grill-me to interrogate the user to make sure the outcome and approach is clear and actionable.

The output should be terse but clearly structured markdown using the template `resources/plan-template.md` (relative to this file) that stands alone.
