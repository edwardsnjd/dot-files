# Add a "local-bin" script

## When to use this skill

Use this skill whenever there is a script worth capturing for general use.

## How to decide

- The script must be useful in future, not just for one project
- The script must have one clear purpose

## How to add a script

- Add the script as an extension-less executable file to `configuration/local-bin/.local/bin/`
- Document the new script in `configuration/local-bin/README.md`
- Read CONTRIBUTING.md for guidance on the script structure in various languages

## Format of README documentation

- Read CONTRIBUTING.md for guidance on the README format
